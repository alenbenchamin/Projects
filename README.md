 Excel Dashboard Project

 📊 Project Overview
This project demonstrates the creation of an interactive dashboard in Microsoft Excel using advanced data visualization and analysis techniques. The dashboard helps users understand trends and insights through various chart types and pivot table analysis.

 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

 📈 Dashboard Features
- Column Chart
- Bar Chart (Pivot Table)
- Line Chart (Pivot Table)
- Area Chart
- Scatter Chart
- Histogram (Statistical Chart)
- Sunburst Chart (Hierarchical Chart)
- Interactive Excel Dashboard Layout

 🎯 Objectives
- Analyze and visualize data effectively.
- Build an interactive dashboard for decision-making.
- Demonstrate Excel data analysis and visualization skills.
